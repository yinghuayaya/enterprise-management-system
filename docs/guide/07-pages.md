## 7.2 登录页 login.html

**访问路径**：`src/pages/login.html`

**作用**：用户登录入口。

### 页面布局图

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│    ┌─────────────────────┬─────────────────────────────────┐   │
│    │                     │                                 │   │
│    │   🌾               │     小麦科技企业管理平台        │   │
│    │                     │                                 │   │
│    │   小麦科技          │     ┌─────────────────────┐    │   │
│    │                     │     │ 用户名              │    │   │
│    │   "让科技更简单"    │     └─────────────────────┘    │   │
│    │                     │                                 │   │
│    │                     │     ┌─────────────────────┐    │   │
│    │                     │     │ 密码                │    │   │
│    │                     │     └─────────────────────┘    │   │
│    │                     │                                 │   │
│    │                     │     [    登录系统    ]          │   │
│    │                     │                                 │   │
│    │                     │     没有账号？立即注册          │   │
│    │                     │                                 │   │
│    └─────────────────────┴─────────────────────────────────┘   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

左侧：品牌展示区（Logo + 宣传语）
右侧：登录表单（用户名 + 密码 + 登录按钮）
```

### 表单字段

| 字段 | 类型 | 验证 |
|------|------|------|
| 用户名 | 文本输入 | 必填 |
| 密码 | 密码输入 | 必填 |

### 交互逻辑

1. 用户输入用户名和密码
2. 点击"登录系统"按钮
3. 前端验证：检查是否为空
4. 调用 `auth.login()` 验证账号
5. 成功：跳转到仪表盘
6. 失败：显示"用户名或密码错误"

**默认账号**：`admin` / `123456`

## 7.3 注册页 register.html

**访问路径**：`src/pages/register.html`

**作用**：新用户注册。

### 表单字段

| 字段 | 类型 | 验证 |
|------|------|------|
| 用户名 | 文本输入 | 必填 |
| 邮箱 | 邮箱输入 | 必填 + 格式验证 |
| 密码 | 密码输入 | 必填 + 至少6位 |
| 确认密码 | 密码输入 | 必填 + 与密码一致 |

### 交互逻辑

1. 用户填写四个字段
2. 点击"立即注册"按钮
3. 前端验证所有字段
4. 调用 `auth.register()` 保存用户信息
5. 成功：显示"注册成功"，跳转到登录页
6. 失败：显示"用户名或邮箱已被使用"
